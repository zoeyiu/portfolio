Project Overview

The fictional client, a leading provider of in-person health services, seeks to expand into the online market with the development of a wellness platform. This website will initially offer freemium services, giving users access to valuable wellness resources at no cost, with plans to introduce a premium subscription model for enhanced features. The platform will provide a variety of wellness resources, including articles, videos, and guides on health topics, along with community support and organized events to foster engagement.

User Story

As a fast-paced agency professional, I want a wellness site with beginner-friendly yoga and stress-relief techniques, so I can easily add mindfulness to my routine and manage the demands of my job for better physical and mental well-being.

As a teacher and single father of two boys, I want a wellness website, having opportunities for social activities to help expand my social circle, as my time is usually spent balancing work and parenting. This would allow me to connect with others.

Frontend Development

Carousel Banner

The banner design goes beyond static images, incorporating videos and carousel slides to enhance the overall visual appeal of the website and attract users. The banner in the homepage contains “Call to Action” buttons to encourage user interaction and guide users to relevant sections of the website.

API Integration & Filter Function & Page Pagination

Event data, including images, dates, categories, and times, is fetched via an API. Default images are provided when none are available in the database to prevent blank spaces. Users can upload their own events, and the content is updated through the API. A filter for event categories was created to help users find events based on their preferences.

Events are displayed using a Card UI design for easy content comprehension. A paginated layout shows 16 cards per page in a 4x4 grid, with JavaScript automatically generating new cards as event data is added. This ensures efficient handling of increasing events. The pagination also updates dynamically when users apply filters, displaying only the relevant cards and recalculating the number of pages needed.

Responsive Design

The website is designed to be fully responsive, ensuring a consistent and user-friendly experience across all devices, from desktops, tablets to mobile phones. The layout adjusts dynamically to different screen sizes, maintaining readability and functionality for all users.

Accessibility

The website implements WCAG guidelines to ensure accessibility for all users, including those with disabilities. Key features include high text contrast for readability, descriptive alt text for all images to assist screen readers, full keyboard navigation for easy site access, and the use of ARIA roles for better structure and navigation. These elements contribute to creating an inclusive user experience.

Readability

To enhance accessibility for users with vision impairments, a font size adjustment feature was implemented. By clicking a button, users can dynamically increase or decrease the font size of the content to suit their individual preferences and needs. The design prioritizes inclusivity, ensuring that all users can navigate the content effectively regardless of their visual abilities.
